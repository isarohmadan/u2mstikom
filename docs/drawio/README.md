## 💡 Tips Editing

### Auto-Layout
```
1. Select All (Ctrl+A)
2. Arrange → Layout → Horizontal Flow
3. Adjust spacing manually
```

### Mengubah Warna
```
1. Select element
2. Format → Fill → White (#FFFFFF)
3. Format → Stroke → Black (#000000), 1px
```

### Export
```
File → Export as → PNG/PDF/SVG
Resolution: 300 DPI (untuk dokumentasi)
```
